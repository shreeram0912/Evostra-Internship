# Evostra Internship Project
This repository documents my work completed during the Evostra Internship Program, focusing on data analytics, business intelligence, and dashboard development using Power BI and data modeling techniques.

---

## 🚀 Project Overview

This project follows a complete data analytics workflow:

- Data understanding and exploration  
- Data modeling using star schema  
- Interactive dashboard development  
- Deployment with security and automation
